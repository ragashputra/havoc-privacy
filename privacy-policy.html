Informasi Umum

Honda Visual On-site Capture ("HAVOC") adalah aplikasi mobile internal yang dikembangkan oleh Ahmad Ragash Putra Dalimunthe dan Tim QCC "Jempol" dan dioperasikan untuk Perusahaan kami yaitu PT Capella Dinamik Nusantara. Aplikasi ini digunakan secara eksklusif oleh karyawan internal perusahaan untuk keperluan operasional pelayanan dalam memvalidasi Data konsumen dengan Surat Pesanan Kendaraan (SPK) atau CDB.

Aplikasi ini tidak tersedia untuk umum dan hanya dapat digunakan oleh karyawan yang telah mendapatkan akun resmi dari perusahaan. Dengan menggunakan Aplikasi, pengguna menyetujui kebijakan privasi ini.

👨‍💻 Pengembang & Pengelola: Ahmad Ragash Putra dan Tim QCC "Jempol"  ·  Website: www.cdn.co.id  ·  Wilayah Operasi: Riau, Kepulauan Riau, Aceh

3

Penggunaan Google Drive API

Aplikasi menggunakan Google Drive API untuk menyimpan file rekaman dan foto secara terorganisir. Berikut penjelasan lengkap penggunaan akses Google yang diminta:

Scope yang DimintaAlasan Dibutuhkan
https://www.googleapis.com/auth/driveMembuat subfolder otomatis berdasarkan nomor mesin, dan mengupload file rekaman/foto ke folder Google Drive milik karyawan
openidVerifikasi identitas akun Google karyawan saat menghubungkan Drive
https://www.googleapis.com/auth/userinfo.emailMenampilkan email Google yang terhubung di halaman pengaturan aplikasi

⚠️ Penting: Aplikasi hanya mengupload file ke Google Drive karyawan. Aplikasi tidak membaca, mengunduh, menghapus, atau mengakses file lain yang sudah ada di Drive karyawan di luar folder yang dibuat oleh aplikasi ini.

Penggunaan data yang diperoleh melalui Google API mematuhi Google API Services User Data Policy, termasuk persyaratan Limited Use.

4

Cara Penggunaan Data

Data yang dikumpulkan digunakan semata-mata untuk keperluan berikut:

  • Autentikasi dan otorisasi karyawan yang menggunakan aplikasi
  • Mengupload file rekaman suara dan foto CDB ke Google Drive karyawan
  • Mengorganisir file dalam subfolder berdasarkan nomor mesin Honda
  • Mencatat riwayat upload untuk keperluan audit internal (disimpan 90 hari, lalu otomatis dihapus)
  • Menampilkan riwayat upload kepada karyawan yang bersangkutan
  • Setiap file rekaman suara dan foto CDB yang sudah diupload ke Google Drive akan dilakukan pengecekan oleh PIC dari Main Dealer, dan file tersebut akan dihapus setelah pengecekan selesai dan dinyatakan valid

📌 Tidak Ada Iklan: Aplikasi ini tidak menampilkan iklan dan data pengguna tidak digunakan untuk keperluan periklanan dalam bentuk apapun.

5

Penyimpanan & Keamanan Data

Data pengguna disimpan dengan ketentuan sebagai berikut:

  • Data akun dan kredensial disimpan di server database yang terenkripsi
  • Password disimpan dalam bentuk hash (bcrypt) — tidak tersimpan dalam bentuk teks biasa
  • Token Google OAuth disimpan terenkripsi dan hanya dapat diakses oleh sistem backend
  • Riwayat upload otomatis dihapus setelah 90 hari
  • Komunikasi antara aplikasi dan server menggunakan protokol HTTPS
  • File yang diupload disimpan di Google Drive milik karyawan — bukan di server perusahaan
  • File rekaman suara dan foto CDB di Google Drive akan diperiksa oleh PIC Main Dealer dan dihapus setelah proses pengecekan selesai dan valid
6

Berbagi Data dengan Pihak Ketiga

PT Capella Dinamik Nusantara tidak menjual, menyewakan, atau membagikan data pengguna kepada pihak ketiga untuk keperluan komersial. Data hanya dibagikan dalam kondisi berikut:

  • Google LLC — sebagai penyedia Google Drive API untuk keperluan upload file (sesuai kebijakan privasi Google)
  • Internal perusahaan — data riwayat upload dapat diakses oleh manajemen terkait untuk keperluan audit operasional
  • Kewajiban hukum — jika diwajibkan oleh peraturan perundang-undangan yang berlaku di Indonesia
7

Hak Pengguna

Sebagai pengguna aplikasi, karyawan memiliki hak berikut:

  • Mengakses dan melihat data profil serta riwayat upload milik sendiri
  • Memperbarui informasi profil kapan saja
  • Memutuskan koneksi Google Drive kapan saja melalui pengaturan aplikasi
  • Mencabut akses aplikasi ke Google Drive melalui halaman Google Account Permissions
  • Menghapus riwayat upload individual melalui fitur di aplikasi
  • Mengajukan permintaan penghapusan akun kepada administrator perusahaan
